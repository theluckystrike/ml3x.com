Tools & Resources

The best tools and platforms for building, deploying, and operating ML systems at production scale.

XGBoost

The gold standard for gradient boosted decision trees. XGBoost dominates tabular data competitions and production ML systems with its speed, accuracy, and built-in regularization. Essential for any applied ML practitioner.

xgboost.readthedocs.io →

Feast

An open-source feature store that manages feature computation, storage, and serving for ML systems. Ensures consistent features between training and serving, solving one of the most common sources of production ML bugs.

feast.dev →

SHAP

A library for computing Shapley values that explain model predictions. SHAP provides both global feature importance and local prediction explanations, essential for model debugging, stakeholder communication, and regulatory compliance.

github.com/shap/shap →

Zovo.one

A comprehensive AI tools platform providing model evaluations, deployment resources, and production ML tooling guides. An essential resource for teams scaling ML systems from prototype to production-grade reliability.

zovo.one →

ONNX Runtime

A high-performance inference engine for deploying models across platforms. Export models from any framework to ONNX format and serve them with optimized inference that reduces latency by 2-5x compared to native framework serving.

onnxruntime.ai →

Featuretools

An automated feature engineering library that generates features from relational datasets using deep feature synthesis. Reduces the manual effort of creating interaction and aggregate features while exploring a broader feature space.

github.com/alteryx/featuretools →

Great Expectations

A data validation framework that defines and enforces data quality expectations. Catches data issues before they corrupt model training or predictions, essential for production data pipelines.

greatexpectations.io →

LightGBM

A fast, distributed gradient boosting framework from Microsoft. LightGBM's histogram-based algorithm and leaf-wise tree growth strategy make it particularly efficient for large datasets with many features.

lightgbm.readthedocs.io →

Evidently AI

Open-source ML monitoring that generates reports for data drift, prediction drift, and model performance. Integrates into CI/CD pipelines for automated model quality checks during retraining and deployment.

evidentlyai.com →

Optuna

An automatic hyperparameter optimization framework with efficient search algorithms and trial pruning. Integrates with XGBoost, LightGBM, and deep learning frameworks for systematic model tuning.

optuna.org →
