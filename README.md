# ML3X

> Free matrix calculator with step-by-step solutions — addition, multiplication, determinants, inverses, and eigenvalues.

**[Use ML3X live](https://ml3x.com/)**

ML3X is a free visual matrix calculator that shows step-by-step solutions for every operation. Add, subtract, and multiply matrices up to 5x5, compute determinants, find inverses, and calculate eigenvalues — all with detailed intermediate calculations displayed on screen. Built for linear algebra students, engineers, and data scientists who want their work shown, not just the final answer.

## Features

- Matrix addition, subtraction, and multiplication with step-by-step breakdown
- Determinant calculator using cofactor expansion (up to 5x5)
- Matrix inverse via adjugate method with full working shown
- Eigenvalue computation with characteristic polynomial display
- Matrix transpose visualization
- Calculation history storing the last 5 operations
- Random fill, clear, and identity matrix quick-setup buttons
- Supports integer, decimal, and negative entries
- Step-by-step explanations suitable for linear algebra homework
- 100% client-side — no data leaves your browser
- MIT licensed
- No signup, no tracking

## Tech Stack

- Pure HTML, CSS, and vanilla JavaScript
- No build step
- No external dependencies (except Google Fonts on some pages)
- Hosted on GitHub Pages with Cloudflare CDN
