Probability Density Function

PDF Formula f(x) = (1 / (σ√(2π))) · e^(-(x-μ)² / (2σ²))

Cumulative Distribution Function

CDF Formula F(x) = 0.5 · [1 + erf((x-μ) / (σ√2))]
